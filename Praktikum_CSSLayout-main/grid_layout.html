Kesimpulan skenario 1 : item-item yang ada di tempat_grid bisa diatur menjadi lebih fleksibel dengan menggunakan grid.

Kesimpulan skenario 2 : Kode program tersebut adalah contoh penggunaan CSS Grid untuk membuat tata letak grid sederhana tiga kolom yang disusun secara inline, menampilkan isi grid nomor 1 sampai 9.

Kesimpulan skenario 3 : Grid gap mengatur jarak setiap item pada tempat_grid. Gap column berarti mengatur jarak pada column, gap row mengatur jarak antar row.

Kesimpulan skenario 4 : dengan grid-gap dapat mengatur sekaligus Column gap dan row gap.
